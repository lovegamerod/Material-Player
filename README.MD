# Material Music Pro 🎵

基于 Material Design 3 和 Apple Music 风格的高级音乐播放器。

## 主要功能
- **全屏沉浸播放**：动态高斯模糊背景。
- **智能歌词**：支持双语歌词合并、自由滚动及点击跳转进度。
- **响应式设计**：适配 PC 与移动端，手机端支持 Bottom Sheet 切换歌单。
- **全局搜索**：快速查找库中所有歌曲。

## 快速使用
1. 将音乐放入 `music/`，歌单分文件夹即可，若想使用歌词显示将歌曲文件同名的lrc文件加入即可。(温馨提示：若歌曲为无损，可以使用git lfs大文件上传)
2. 编辑 `data.json` 配置路径（可以使用generate_data_json.py快速生成）。
3. Enjoy Music Time!

（悄咪咪：为什么叫Material Player是因为原本打算用Material Design 3设计的）
